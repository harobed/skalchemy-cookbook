SKAlchemy Cookbook¶

Voici le cookbook SQLAlchemy de Stéphane Klein.

Au fil du temps, j’ai accumulé un certain nombre de classes “helper”, de classes Mixin ainsi que des classes modèles assez génériques.
Pendant longtemps, j’ai passé mon temps à rechercher le code de ces classes dans le code source de mes précédents projets.
J’ai fini par trouver cela fastidieux et j’ai donc décidé d’écrire petit à petit ce Cookbook.

J’espère aussi qu’il pourra vous être utile… et surtout, n’hésitez pas à forker ce projet pour m’envoyer vos Mixin, vous classes modèles génériques…
